## Getting Started

### Logging In
1. Visit the Bramhollow portal website
2. Click **"Log In"** button
3. Use your Replit account to authenticate
4. You'll be redirected to your personalized dashboard

### First Time Setup
- Your account will be automatically created upon first login
- Contact a manager or administrator to assign your proper role and unit information
- Update your profile information in the dashboard

## Resident Features

### Dashboard Overview
Your dashboard provides quick access to:
- Recent announcements
- Your active service requests
- Upcoming community events
- Available amenity reservations

### Viewing Announcements
- **General**: Community updates and information
- **Urgent**: Time-sensitive notices requiring immediate attention
- **Newsletter**: Quarterly community newsletters
- **Maintenance**: Building maintenance schedules
- **Financial**: Budget updates and financial notices

### Submitting Service Requests
1. Go to **"Service Requests"** tab
2. Click **"New Request"**
3. Fill out the form:
   - **Title**: Brief description of the issue
   - **Category**: Select appropriate category (maintenance, repair, etc.)
   - **Priority**: Low, Medium, or High
   - **Description**: Detailed explanation of the problem
4. Submit and track status updates

### Community Events
- View upcoming events in the **"Events"** tab
- RSVP to events you plan to attend
- Check event details including date, time, and location

### Amenity Reservations
- Reserve community facilities like:
  - Meeting rooms
  - Recreation areas
  - Guest parking spaces
- Check availability and book time slots

## Admin Panel (Managers & Administrators)

### Accessing Admin Features
Managers and administrators will see an additional **"Admin"** tab in their dashboard.

### Announcement Management

#### Creating Announcements
1. Navigate to **Admin → Announcements**
2. Click **"Create Announcement"**
3. Fill out the form:
   - **Title**: Clear, descriptive headline
   - **Type**: Select from General, Urgent, Newsletter, Maintenance, Financial
   - **Priority**: Low, Medium, High (affects display prominence)
   - **Content**: Full announcement text (supports formatting)
   - **Publish Date**: When announcement goes live
   - **Expiration Date**: When announcement is automatically hidden

#### Managing Existing Announcements
- View all announcements in a sortable table
- Edit announcements by clicking the edit icon
- Monitor publish/expiration dates
- Track announcement engagement

### Service Request Management

#### Viewing All Requests
- Access complete list of resident service requests
- Filter by status, priority, or category
- Sort by submission date or last update

#### Updating Request Status
1. Locate the service request
2. Use the status dropdown to change status:
   - **Pending**: Initial submission state
   - **In Progress**: Work has begun
   - **Completed**: Issue resolved
   - **Cancelled**: Request no longer needed

#### Request Categories
- **Maintenance**: Building upkeep and repairs
- **Security**: Safety and access issues
- **Amenities**: Community facility problems
- **Administrative**: Paperwork and procedural requests

### Unit Management

#### Adding New Units
1. Go to **Admin → Units**
2. Click **"Add Unit"**
3. Enter unit details:
   - **Unit Number**: Unique identifier
   - **Floor**: Building level
   - **Square Footage**: Unit size
   - **Bedrooms/Bathrooms**: Count
   - **Parking Spaces**: Assigned parking
   - **Balcony**: Yes/No

#### Managing Existing Units
- Edit unit information as needed
- Track occupancy status
- Manage resident assignments

## Service Requests

### For Residents

#### Submitting Effective Requests
1. **Be Specific**: Clearly describe the issue
2. **Include Details**: Unit number, exact location of problem
3. **Set Appropriate Priority**:
   - **High**: Safety issues, water leaks, security problems
   - **Medium**: Non-emergency repairs, noise complaints
   - **Low**: Cosmetic issues, suggestions

#### Tracking Your Requests
- View all your requests in the Service Requests tab
- Monitor status changes
- Receive notifications when status updates

### For Administrators

#### Request Categories
- **Maintenance**: HVAC, plumbing, electrical, general repairs
- **Security**: Access control, lighting, surveillance
- **Amenities**: Pool, gym, meeting rooms, parking
- **Administrative**: Documents, payments, policy questions

#### Response Time Guidelines
- **High Priority**: 24 hours
- **Medium Priority**: 3-5 business days
- **Low Priority**: 1-2 weeks

## Troubleshooting

### Login Issues
**Problem**: Cannot access the system
**Solution**: 
1. Ensure you're using the correct Replit account
2. Contact an administrator to verify your account setup
3. Clear browser cache and try again

### Missing Features
**Problem**: Don't see admin panel or expected features
**Solution**: 
1. Check your user role with an administrator
2. Log out and log back in to refresh permissions
3. Contact management for role assignment

### Service Request Problems
**Problem**: Cannot submit or view service requests
**Solution**:
1. Refresh the page and try again
2. Ensure all required fields are completed
3. Contact technical support if issues persist

### Newsletter Access
**Problem**: Cannot access newsletter creation tools
**Solution**:
1. Verify you have manager or administrator role
2. Contact system administrator for permission updates

### General Technical Issues
**Problem**: System errors or unexpected behavior
**Solution**:
1. Try refreshing the page
2. Clear browser cache and cookies
3. Try using a different browser
4. Contact technical support with error details
