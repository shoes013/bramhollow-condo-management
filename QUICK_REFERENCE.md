## Common Tasks

### For Residents

#### Submit Service Request
1. **Service Requests** tab → **New Request**
2. Fill out: Title, Category, Priority, Description
3. Submit and track status

#### View Announcements
1. **Dashboard** → View recent announcements
2. **Announcements** tab → See all announcements
3. Look for priority indicators (High/Medium/Low)

#### RSVP to Events
1. **Events** tab → View upcoming events
2. Click event → **RSVP**
3. Confirm attendance

### For Managers/Administrators

#### Create Announcement
1. **Admin** tab → **Announcements**
2. **Create Announcement**
3. Set: Title, Type, Priority, Content, Dates
4. **Publish**

#### Update Service Request
1. **Admin** tab → **Service Requests**
2. Find request → Update status dropdown
3. Select: Pending/In Progress/Completed/Cancelled

#### Create Newsletter
1. **Newsletter** tab
2. Write content in markdown editor
3. Select template → **Export HTML**
4. Import to Microsoft 365

#### Add New Unit
1. **Admin** tab → **Units**
2. **Add Unit**
3. Fill: Unit Number, Floor, Details
4. **Save**

## Emergency Procedures

### High Priority Issues
- Safety hazards
- Water leaks
- Security breaches
- Power outages affecting common areas

### Steps:
1. **Immediate**: Submit HIGH priority service request
2. **If urgent**: Call emergency maintenance: [Emergency Number]
3. **If life-threatening**: Call 911 first, then notify management
