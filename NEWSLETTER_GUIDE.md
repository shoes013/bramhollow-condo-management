# Newsletter Creation Guide - Bramhollow Condominium

*Quarterly Newsletter System for Microsoft 365 Distribution*

## Overview

The newsletter system allows managers and administrators to create professional, Microsoft 365-compatible newsletters using an easy markdown editor with real-time preview.

---

## Getting Started

### Access Requirements
- Manager or Administrator role required
- Log into the Bramhollow portal
- Navigate to the **Newsletter** tab in your dashboard

### System Features
- **Markdown Editor**: Easy text formatting
- **Real-time Preview**: See your newsletter as you type
- **Multiple Templates**: Professional email layouts
- **Export Options**: HTML and plain text formats
- **Mobile Preview**: Check appearance on different devices

## Export and Distribution

### Export Options

#### HTML Export
1. Complete your newsletter content
2. Select your preferred template
3. Click **"Export HTML"**
4. Save the file to your computer
5. The file will be named: `bramhollow-newsletter-[date].html`

#### Copy to Clipboard
1. After selecting template, click **"Copy HTML"**
2. Paste directly into Microsoft 365 email
3. The formatting will be preserved
4. Ready to send to your mailing list

#### Plain Text Export
1. Click **"Export as Text"**
2. Creates a simple text version
3. Useful for accessibility
4. Backup format for email issues

### Microsoft 365 Distribution

#### Step-by-Step Process
1. **Create newsletter** in the portal
2. **Export as HTML** or copy to clipboard
3. **Open Microsoft 365 Outlook**
4. **Create new email**
5. **Paste or import** your newsletter content
6. **Add your recipient list**
7. **Send test email** to verify appearance
8. **Distribute to community**

#### Email Settings Recommendations
- **Subject**: "Bramhollow Newsletter - [Month] [Year]"
- **Priority**: Normal (unless urgent updates)
- **Format**: HTML (automatic with our export)
- **Reply-to**: Management email address

## Content Guidelines

### Writing Style
- **Professional but friendly** tone

- **Proofread carefully**

### Content Standards
- **Accuracy**: Double-check all dates, numbers, and contact info
- **Relevance**: Include information that affects residents

## Troubleshooting

### Common Issues

#### Formatting Problems
**Issue**: Markdown not displaying correctly
**Solution**: 
- Check markdown syntax
- Use preview to verify formatting
- Try simpler formatting if complex syntax fails

#### Export Issues
**Issue**: HTML export not working
**Solution**:
- Refresh the page
- Try copying to clipboard instead
- Use plain text export as backup

#### Email Display Problems
**Issue**: Newsletter looks wrong in Outlook
**Solution**:
- Test with different templates
- Use simpler formatting
- Send test emails before full distribution

#### Mobile Appearance
**Issue**: Newsletter hard to read on phones
**Solution**:
- Use mobile preview mode
- Keep paragraphs short
- Avoid complex layouts

### Getting Help
- **Technical Issues**: Contact system administrator
- **Content Questions**: Consult board members
- **Distribution Problems**: Check with management company
